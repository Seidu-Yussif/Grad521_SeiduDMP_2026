# Data description
The project uses two related datasets: Micro-CT scans and derived segmentation data for training deep learning models. The Micro-CT scans are 3D images that show biofilms inside porous materials, collected under different imaging conditions for both speed and quality. A CT scan (computed tomography scan) is an imaging technique that uses X-rays taken from multiple angles to reconstruct detailed cross-sectional and 3D images of an object’s internal structure.

# Roles and Responsibilities
In this research project, data management responsibilities are shared mainly between the Principal Investigator (PI), the graduate student, and the department’s network administrator. The PI is responsible for implementing the Data Management Plan (DMP) and ensuring that all data practices follow project and institutional guidelines.
The graduate student (myself) is responsible for most of the data-related tasks. This includes data collection, data processing, metadata generation, and data analysis. The raw data in this project consists of X-ray projection images from micro-CT imaging. After collecting the raw data, I transfer it to an online reconstruction server where it is processed into 3D volume data. 

# Data Standards and Metadata
Because the data does not involve human subjects or sensitive information, there are no specific privacy regulations such as HIPAA or FERPA. However, the project is required to make data publicly available after publication. In compliance with DOE policies, the final processed datasets and associated metadata will be shared in a public repository. 

# Storage and Security 
Data is stored across several locations during the project. Raw data is first stored on the acquisition system and then transferred to an online reconstruction server. After processing, reconstructed data is stored on a high-capacity storage server managed by the school. Additionally, Box is used for sharing and archiving important datasets, including intermediate and final processed data. Access to these systems is restricted to authorized users only, ensuring data security.

# Access and data sharing
Data will be deposited in the Digital Porous Media repository upon publication of research. 
# Archiving and Preservation
To ensure long-term access, all final processed data is archived in Box with proper organization and naming conventions
